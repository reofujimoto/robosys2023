## plusコマンド 

標準入力から読み込んだ数字を足す。

### 入力例

1~5の整数を
```
$ seq 5 | ./plus
```

### 出力
```
15
```


## ダウンロード方法
```
$ git clone git@github.com:reofujimoto/robosys2023.git
$ cd robosys2023/
```
